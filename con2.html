    ᠌

  • -наличие широкого ассортимента шаблонов;

    • ᠌

  • -по своему желанию можно создать уникальный шаблон;

    • ᠌

  • -можно легко создавать мобильный вариант сайта;

    • ᠌

  • -гибкость настроек;

    • ᠌

  • -высокую мощность редактора;

    • ᠌

  • -предусмотрена функция Drag & Drop;

    • ᠌

  • -отсутствие проблем с продвижением проекта;

    • ᠌

  • -интерфейс ресурса русифицирован и интуитивно понятный даже несмотря на то, что он иностранный;

    • ᠌

  • -возможность создания бизнес-страницы для Facebook;

    • ᠌

  • -техподдержка достаточно отзывчивая;

    ᠌

  • -если версия бесплатная, то адрес сайта будет трудно запоминающимся;

  • -скорости работы проекта также будет ограничена.

    Онлайн-сервис Setup

    Данный конструктор отлично подходит для начинающих заниматься бизнесом в интернете. Портал предлагает свыше 5000 вариантов легко настраиваемых шаблонов.

    ᠌

    В виде бонуса сервис предложит домен в зоне ru бесплатно, а также поможет продвигать проект по НЧ запросам.

    Среди плюсов конструктора Сетап:

    ᠌

  • -красивые шаблоны;

    • ᠌

  • -возможность раскрутки проекта по низкочастотникам;

    • ᠌

  • -интеграция с площадкой SeoPult;

    • ᠌

  • -хорошая техподдержка.

    • ᠌

    Присутствуют и серьезные недостатки (в основном для бесплатной версии):

    ᠌

  • нет возможности переезда проекта на другой хостинг;

    • ᠌

  • -чтобы получить бесплатный домен потребуется создать около 20 станиц, а затем отправить на модерацию. Это не дает 100%-ной гарантии того, что заявка будет одобрена;

    • ᠌

  • -бесплатная версия ресурса ограничивается малым дисковым пространством (100 мб);

    • ᠌

  • -внешние ссылки не будут индексироваться поисковиками.

    • ᠌

    Следовательно, Setup будет идеальной площадкой для начинающих в связи с простотой и удобством.
